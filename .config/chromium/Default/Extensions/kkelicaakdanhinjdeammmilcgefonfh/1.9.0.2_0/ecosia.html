This extension supports Ecosia.org by adding EcoLinks to the Google search page. You can opt-out of this feature in the options, if you want to.

EcoLinks Settings

Please note that this setting does not alter in any way your search results. It just takes the links returned by Google and converts them to EcoLinks, without adding any extra content.
It is a completeley transparent and free way for you to plant trees and support this extension at the same time. :) Learn more
